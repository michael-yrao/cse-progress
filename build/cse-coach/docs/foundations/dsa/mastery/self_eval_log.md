# Self-Evaluation Log

Append-only record of corrections the learner gives the coach. Once a week the
coach reviews this log and promotes recurring corrections into standing behavior
(see the cse-coach skill, §8). This is how the coach gets sharper for *this*
learner over time.

Format: one dated line per correction.

```
- YYYY-MM-DD — [what the coach did] → [the correction] → [rule to apply going forward]
```
